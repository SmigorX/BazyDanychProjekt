# Projekt na bazy danych - geolokalizacyjne notatki.

Projekt w założeniach ma umożliwić na dzielenie się wiadomościami w geolokalizacyjnym kontekście.
Dzielenie się notatkami z innymi oraz zakładanie grup zrzeszających uczestników.

> [!NOTE]
> Po uruchomieniu backendu pod ścieżką `/docs` znaleźć można interaktywną dokumentację API w postaci swaggera.

> [!IMPORTANT]
> dokumentacja_api.html jest statyczną wersją wygenerowaną na podstawie swagger (patrz "Note") i może być nieauktualna.
